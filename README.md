# AI智有方 · 内容创作技能库

AI 时代的内容生产流水线 — 从原始素材到交付物，结构先行、文案落地、用户确认、按需输出。

## 技能清单

| 技能 | 用途 | 输出形式 |
|---|---|---|
| [aidai-ppt-skill](./aidai-ppt-skill/) | 长文/报告/想法 → PPT 交付 | HTML 幻灯片 / 传统 PPT 大纲 / 视频脚本 / 演讲提词稿 / 信息图文案 |

## 流水线标准流程

所有 skill 共用一套四阶段工作流：

```
源文档 → [提取结构] → [润色文案] → [用户确认] → [渲染输出]
```

1. **骨架确认（Step 0a）** — 跟用户定标题结构、页数、风格基调
2. **内容生成** — content-to-slides 提取 → humanizer-zh 润色
3. **内容确认（Step 2.5）** — 完整文案呈现给用户审阅，改到位再往下
4. **按格式渲染** — HTML / PPT 大纲 / 视频脚本 / 其他

**原则：** 不跳过人工确认，不替用户做文案决策，不默认输出格式。
